# OCA

## Kohonen

### Dependencies

* Python 3.11.5
* Argparse
* Numpy
* Matplotlib

### Run

in kohonen folder:

```sh
python main.py input.txt
```

or

```sh
python main.py --help
```

### Documentation

```sh
kohonen/docs/_build/html/index.html
```

or

```sh
kohonen/docs/_build/latex/SOM_Documentation.pdf
```

in bfs folder:

```sh
python main.py input.txt
```
or

```sh
python main.py --help
```

### Documentation

```sh
bfs/docs/_build/html/index.html
```

or
